  ШАГ 4: ПРОСМОТР ИНФОРМАЦИИ ОБ УСТРОЙСТВЕ
═══════════════════════════════════════════════════════════════

1. Кликните на строку с устройством (например телефон)
2. Нажмите "Подключить" (на самом деле - "Информация")
3. Увидите:
   - Имя устройства
   - MAC адрес
   - Тип
   - Статус (подключено/сопряжено)

═══════════════════════════════════════════════════════════════
  ШАГ 5: ОТПРАВКА ФАЙЛА НА ТЕЛЕФОН
═══════════════════════════════════════════════════════════════

1. Выберите устройство (телефон) в таблице
2. Нажмите "Отправить файл"
3. Выберите файл (MP3, фото, видео - любой)
4. Откроется "Windows Bluetooth File Transfer Wizard"
5. В списке выберите ваш телефон
6. Нажмите "Далее"
7. На телефоне появится уведомление - ПРИМИТЕ!
8. Дождитесь завершения передачи
9. ✓ Файл на телефоне!

═══════════════════════════════════════════════════════════════
  ШАГ 6: ВОСПРОИЗВЕДЕНИЕ НА ТЕЛЕФОНЕ
═══════════════════════════════════════════════════════════════

Android:
  1. Откройте уведомление о полученном файле
  2. Нажмите на файл
  3. Выберите плеер (если MP3)
  4. Файл воспроизведется!

Или:
  1. Откройте "Файлы" или "Мои файлы"
  2. Перейдите в "Bluetooth" или "Загрузки"
  3. Найдите файл
  4. Откройте его

═══════════════════════════════════════════════════════════════
  ВОЗМОЖНЫЕ ПРОБЛЕМЫ
═══════════════════════════════════════════════════════════════

ПРОБЛЕМА: Устройства не находятся при сканировании
  ПРИЧИНА: Устройства уже подключены (не в режиме обнаружения)
  РЕШЕНИЕ: Нажмите "Обновить адаптер" - увидите подключенные

ПРОБЛЕМА: "Сопряжено но не подключено"
  ПРИЧИНА: Нормально! Устройство знакомо системе
  РЕШЕНИЕ: Это НЕ проблема - отправка файлов работает

ПРОБЛЕМА: Телефон не принимает файл
  ПРИЧИНА: Bluetooth выключен или телефон не видим
  РЕШЕНИЕ: 
    1. На телефоне: Настройки → Bluetooth → ВКЛ
    2. Сделайте телефон видимым
    3. Попробуйте снова

ПРОБЛЕМА: fsquirt не запускается
  ПРИЧИНА: Проблема с Windows
  РЕШЕНИЕ: Альтернатива - ПКМ на файл → Отправить → Bluetooth

СЦЕНАРИЙ (5 минут):

1. Запуск Lab6 (30 сек)
   → Показать: адаптер найден, логи создаются

2. Сканирование (1 мин)
   → Показать: процесс сканирования, найденные устройства
   → Или "Обновить адаптер" для подключенных

3. Информация об устройстве (30 сек)
   → Выбрать телефон → "Подключить"
   → Показать: имя, MAC, статус

4. Отправка файла (2-3 мин)
   → "Отправить файл" → Выбрать MP3
   → Откроется мастер Windows
   → Выбрать устройство → Далее
   → На телефоне принять файл
   → ✓ Файл передан!

5. Логи (30 сек)
   → Показать папку логов
   → Открыть bluetooth_log.txt
   → Показать детали всех операций
